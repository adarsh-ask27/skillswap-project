# SkillSwap – Skill Exchange Platform

SkillSwap is a modern web platform that allows users to exchange skills without paying money.  
Users can teach one skill and learn another skill from other users.

This project is built using HTML, CSS and JavaScript with advanced UI design and animations.

## 💡 How It Works

1. User creates profile
2. User enters skills offered and skills required
3. System finds matching users
4. Users share knowledge using chat, lessons, and resources
5. Users track progress and complete learning
